# Contributing to Awesome Solopreneur AI

Thank you for helping make this the definitive AI resource for solopreneurs. This list only stays useful if its entries stay accurate, relevant, and high-quality. These guidelines keep the bar high.

## The Standard

Every tool on this list should meet a simple test: **Would a non-technical solopreneur actually use this tool to run their business?**

If the answer is yes, it belongs here. If it only makes sense for developers, enterprise teams, or academics, it doesn't.

## Before You Submit

Check that your tool meets all of the following criteria:

### ✅ Must-Have Requirements

- **Real and publicly available.** The tool must be accessible at the URL you provide. No waitlists, private betas, or vaporware.
- **Actively maintained.** The last update should be within the past 12 months. If a product is abandoned, it shouldn't be on this list.
- **Accessible to non-developers.** Tools that require coding ability to use should be noted as such. Tools that require no coding are preferred.
- **Accurate pricing.** Free, Freemium, Paid, or Open Source. If the pricing has changed, update it.
- **Unique value.** Don't add a tool that does the exact same thing as three tools already on the list unless it's meaningfully better, cheaper, or different.

### ❌ Do Not Submit

- Tools you haven't personally used or thoroughly researched
- Tools with no visible pricing or unclear business models
- Tools that are primarily useful to developers or data scientists (unless in the Developer Tools section)
- Affiliate links or referral codes embedded in URLs
- Your own tool without disclosing that you built it (disclosure is fine, it should just be clear)

## How to Submit a New Tool

### Format

All entries follow this format:

```
- [Tool Name](https://tool-url.com) - One sentence that explains what it does and why a solopreneur would use it. `Pricing`
```

**Pricing tags:** Use exactly one of: `` `Free` `` / `` `Freemium` `` / `` `Paid` `` / `` `Open Source` ``

- **Free** — Core product is completely free with no paid tiers
- **Freemium** — Free tier exists with meaningful functionality; paid tiers unlock more
- **Paid** — No free tier (trial periods are fine, but ongoing free tier is required for Freemium)
- **Open Source** — Source code is publicly available; may have hosted paid plans

### Description guidelines

**Do write:**
> Automatically transcribes meeting recordings and generates summaries, action items, and follow-up emails.

**Don't write:**
> The world's most advanced AI meeting platform with cutting-edge natural language processing.

Keep descriptions factual and specific. What does it do? Why would a solopreneur use it instead of something else?

### Which section?

Add your tool to the most appropriate section. If it spans multiple categories, put it in the primary one and note secondary use cases in the description.

## How to Submit a Pull Request

1. **Fork** the repository
2. **Create a branch** named `add-[toolname]`
3. **Add your entry** in the correct section, maintaining alphabetical order within sections
4. **Update the tool count** in the README header if applicable
5. **Open a PR** with a title like `Add [Tool Name] to [Section]`
6. **In the PR description**, include:
   - Why this tool belongs on the list
   - Your experience with it (or your source)
   - Current pricing

## Updating Existing Entries

If a tool has changed its name, URL, pricing, or description, submit a PR with the updated information. Include a brief note in the PR explaining what changed and why.

## Removing Tools

If a tool has shut down, gone significantly downhill, or is no longer accessible, open a PR to remove it. Include evidence (e.g., their shutdown announcement or a 404 page).

## Reporting Issues

Not comfortable with Git? Open an [Issue](https://github.com/jakeolschewski/awesome-solopreneur-ai/issues) with:
- The tool you want to add, update, or remove
- The reason
- Any relevant links

## Disclosure Policy

If you work for, have an affiliation with, or have a financial interest in a tool you're submitting, you must disclose that in your PR description. We review everything, but transparency helps.
